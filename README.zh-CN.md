# she_wiki

[English](./README.md)

这是一个按三层结构整理的本地 wiki：

1. `a_summaries`
2. `b_extractions`
3. `c_sources`

编号规则：

- 层级使用字母前缀，按阅览顺序排序：`a / b / c`
- `a` 用于 summaries，`b` 用于 extractions，`c` 用于 sources
- 后续数字直接使用自然数序列，例如 `a1`、`a2`、`b3`、`c9`
- `b` 层的文件继承所属分类前缀，例如 `b3` 分类下的文件使用 `b31`、`b32`、`b33`
- 目录和文件都按自然数往后递增即可，不需要固定四位数字
