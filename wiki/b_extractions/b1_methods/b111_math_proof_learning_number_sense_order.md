# 数学证明学习的数感优先顺序

## 提取结果

- 数学证明学习里，最先该训练的不是“逐行复述证明”，而是对条件信息增量的敏感度。
- 证明阅读效率低常见原因不是意志力不足，而是缺失了“这个条件为什么重要”的前置语义层。
- 更稳定的学习路径是：先判断方向，再做形式化校验，而不是一上来全程硬啃形式化文本。
- 课堂讲解、讨论班和同伴交流的重要价值，在于补足教材中被压缩掉的中间推理步骤。

## 稳定结构

1. 先识别命题中的条件 `p` 是否引入了新的结构约束或信息增量。
2. 基于该信息增量先做方向性判断：它可能导致哪些结论形态。
3. 再进入 `p => q` 的形式化证明，检查直觉与推导是否一致。
4. 对不一致位置回溯：是条件理解偏差、定义遗漏，还是推理链缺环。
5. 用讨论和讲解补齐“教材省略的中间层”，再回读证明细节。
6. 把“信息增量判断 -> 方向预测 -> 形式化验证”固化为重复训练循环。

## 可直接复用

- 读定理前先写一句话：这个条件到底比无条件情形多了什么约束。
- 卡在证明细节时，先暂停问“这一行在服务哪个全局方向”，再继续推演。
- 复盘时不要只记结论和技巧，优先记录“我最初错判了哪个信息增量”。
- 听课或讨论时主动追问“为什么这个条件是关键”，不要只问“这步怎么算”。
- 备考或自学排程里，证明阅读和语义解释应交替安排，而不是单一路径堆时长。
